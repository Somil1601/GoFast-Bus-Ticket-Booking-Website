# 🚌 GoFast – Bus Ticket Booking Website

A responsive bus ticket booking website developed using **HTML**, **CSS**, and **JavaScript** as part of my **ICT course project**. The website provides a user-friendly interface for exploring bus routes, viewing terminal information, and simulating the ticket booking process.

## 📖 Overview

GoFast is a front-end web application designed to simulate an online bus ticket booking system. The project focuses on responsive web design, intuitive navigation, and interactive user experience using HTML, CSS, and JavaScript.

This project was developed as part of my first-year ICT coursework and helped me strengthen my understanding of front-end web development.
