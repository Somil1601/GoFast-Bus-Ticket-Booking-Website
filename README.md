# 🚌 GoFast – Bus Ticket Booking Website

## 📖 Overview

GoFast is a responsive bus ticket booking website developed as an ICT course project. It allows users to explore routes, view bus information, and book tickets through an interactive and user-friendly interface.

---

## ✨ Features

- Responsive homepage
- Route information
- Bus terminal details
- Ticket booking page
- Attractive user interface
- Interactive JavaScript functionality

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
